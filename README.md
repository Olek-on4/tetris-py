# Tetris Game

This repository contains a very old Tetris engine implemented in Python using the `tkinter` library. It was created as a personal project to explore object-oriented programming (OOP) and GUIhistorical development in Python. This is here just for historical evidence and as a memento to my future self.

## Features
- Core Tetris gameplay mechanics.
- Visual representation of Tetris pieces on a `tkinter` canvas.
- Message communication system for in-game notifications.

## File Overview
- **Tkinter_tetris_v1.1.py**: Main implementation file for the Tetris game logic.
- **tkinter_extension.py**: Contains the `Message` class for managing in-game messages.
- **t_icon.ico**: Icon file for the application.

## How to Run
1. Ensure you have Python installed (version 3.6 or higher).
2. Run the `Tkinter_tetris_v1.1.py` file:
   ```bash
   python Tkinter_tetris_v1.1.py
   ```

## Dependencies
- Python 3.6+
- `tkinter` (included with most Python installations)
